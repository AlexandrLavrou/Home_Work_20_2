# 🎬 Movie Library API

A Flask-based REST API for managing movies, genres, directors, users, and favorites. Supports JWT authentication and admin-restricted actions, with full Swagger documentation.

---

## 🚀 Features

- ✅ User Registration and Login (JWT-based)
- 🎥 CRUD operations for Movies
- 🎭 CRUD operations for Genres
- 🎬 CRUD operations for Directors
- ⭐ Manage Favorite Movies per User
- 📑 Swagger UI Documentation

---

## ⚙️ Tech Stack

- Python 3.12
- Flask + Flask-RESTx
- SQLAlchemy
- Alembic (migrations)
- Marshmallow (serialization)
- Pytest (testing)

---

## 📁 Project Structure

├── dao/ # Data Access Objects (SQLAlchemy models & queries)

├── service/ # Business logic layer

├── views/ # Flask-RESTx endpoints

├── tests/ # Pytest test suite

├── migrations/ # Alembic database migrations

├── setup_db.py # SQLAlchemy setup

├── main.py # App entry point

└── config.py # App configuration

```bash

python -m venv venv

🛠️ Database Setup

Alembic Migrations
Initialize Alembic (first time only):
flask db init
Generate migration:
flask db migrate -m "Initial migration"
Apply migration:
flask db upgrade
▶️ Running the App

python main.py

🔐 Auth Flow

Register
POST /auth/register
{
  "email": "user@example.com",
  "password": "1234"
}
Login
POST /auth/login
{
  "email": "user@example.com",
  "password": "1234"
}
Response:

{
  "access_token": "....",
  "refresh_token": "...."
}
Use header:

Authorization: Bearer <access_token>
🌟 Example Endpoints

GET /movies/ - List movies (with filters, pagination)
POST /movies/ - Create movie (admin)
GET /movies/<id> - Get single movie
PUT /movies/<id> - Update movie (admin)
DELETE /movies/<id> - Delete movie (admin)
GET /genres/
GET /genres/<id>
GET /directors/
GET /directors/<id>
GET /favorites/movies - List user favorites
POST /favorites/movies/<movie_id> - Add to favorites
DELETE /favorites/movies/<movie_id> - Remove from favorites
